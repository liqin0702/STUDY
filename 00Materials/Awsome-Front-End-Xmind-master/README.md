### 参与维护 Contributing
> 如果你也喜欢通过使用思维导图的方式来辅助学习，欢迎将整理的思维导图或则是读书笔记贡献出来，或则与我联系微信：646321933，备注github

1. fork 该项目，克隆到你本地
2. 在项目中找到对应的前端分类，例如 Node（如果没有相关目录可以新建一个新分类，或则与我联系）
3. 在项目中添加新的思维导图
4. 发起 pull request 
5. 在项目issue里添加对应的读书笔记或则技术文章,我会同步到我的前端知乎专栏
